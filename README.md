## Overview

### Stack

- Next.js 16 (Turbopack)
- React 19
- Tailwind CSS v3
- Radix UI + shadcn-style (CVA, Tailwind)

### Featured

- Clean & modern design
- Light/Dark themes (next-themes)
- Framer Motion animations
- SEO optimized (metadata, Open Graph)
- Vercel Analytics
- Projects page; nav for Experience, TIL, and Blogs


## Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

### Environment

Copy `.env.example` to `.env.local` and add any required keys (e.g. `NEXT_PUBLIC_NASA_API_KEY` for the NASA APOD widget).
